# EncodingFormat

The format to return the embeddings in. Can be either `float` or [`base64`](https://pypi.org/project/pybase64/).

## Example Usage

```typescript
import { EncodingFormat } from "argot-open-ai/models/components";

let value: EncodingFormat = "float";
```

## Values

```typescript
"float" | "base64"
```
