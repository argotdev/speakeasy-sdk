# OpenAIFile

The `File` object represents a document that has been uploaded to OpenAI.

## Example Usage

```typescript
import { OpenAIFile } from "argot-open-ai/models/components";

let value: OpenAIFile = {
  id: "<id>",
  bytes: 725255,
  createdAt: 501324,
  filename: "example.file",
  object: "file",
  purpose: "vision",
  status: "processed",
};
```
