# VectorStoreFileObject

A list of files attached to a vector store.

## Example Usage

```typescript
import { VectorStoreFileObject } from "argot-open-ai/models/components";

let value: VectorStoreFileObject = {
  id: "<id>",
  object: "vector_store.file",
  usageBytes: 829898,
  createdAt: 968287,
  vectorStoreId: "<id>",
  status: "failed",
  lastError: {
    code: "server_error",
    message: "<value>",
  },
};
```
